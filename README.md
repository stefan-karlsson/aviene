# Aviene

A hobby project focused on developing both foundational building blocks and complete solutions for Software as a Service (SaaS). This repository encompasses core infrastructure components, reusable services, and full-featured applications - all designed to explore and implement modern SaaS architecture patterns and best practices.

## Architecture Decision Records

[![ADR](https://stefan-karlsson.github.io/aviene/adr/badge.svg)](https://stefan-karlsson.github.io/aviene/adr/)

This repository uses [Log4brains](https://github.com/thomvaill/log4brains) to manage Architecture Decision Records (ADRs). ADRs are used to document important architectural decisions, their context, and consequences.

You can explore the ADRs in a user-friendly format via the [ADR web interface](https://stefan-karlsson.github.io/aviene/adr/).

To create a new ADR, run:

```sh
pnpm adr:new
```
